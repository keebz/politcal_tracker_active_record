Political Tracker
=================

by [Lee Kebler] (https://github.com/keebz)

August 22, 2014
_______________

###About
This is a political tracker system that will allow a person to add, list, and sort politicians by their state and funding.
###Setup
Either clone this repository or download the .zip and open in your console or sublime. You will have to create your own databases. Run bundler, then run rake db:create.
